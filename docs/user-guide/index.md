# User Guide

The dashboard has one page per analysis domain. This guide explains what each page shows and how to interpret it.

## Navigation

| Page | What it answers |
|------|-----------------|
| [Home](home.md) | How healthy are my finances overall? What should I do first? |
| [Cashflow](cashflow.md) | Where does my money go? Do I have surplus to deploy? |
| [Debt](debt.md) | Which debts should I pay off first? How long until I'm debt-free? |
| [Goals](goals.md) | Are my goals reachable? What would it take to hit them? |
| [Investments](investments.md) | Is my portfolio on track? Will my pension last? |
| [Mortgage](mortgage.md) | Can I afford a house? What LTV should I target? |
| [Life Events](life-events.md) | What does my financial life look like year-by-year? |
| [Scenarios](scenarios.md) | What happens if things go wrong? |
| [Scores & Grades](scores.md) | What do the numbers actually mean? |

## How the dashboard is organised

Every page follows the same pattern:

1. **Headline metric** — the single most important number on this page
2. **Breakdown** — supporting detail that explains the headline
3. **Recommendations** — what to do about it, ranked by impact

If you only have 30 seconds, read the headline and the top recommendation. The rest is for when you want to understand why.

## Reading the numbers

- **Monetary values** are in pounds sterling.
- **Percentages** with no qualifier are annual unless explicitly monthly.
- **Real values** are adjusted for inflation (today's pounds).
- **Nominal values** are future pounds at face value.
- **Grades** run A+ (90+), A (80-89), B (70-79), C (60-69), D (50-59), F (<50).

## Colour coding

GroundTruth uses a colour-blind safe palette:

- **Green** — on track, affordable, healthy
- **Amber** — warning, at risk, tight
- **Red** — unaffordable, unreachable, high risk
- **Grey** — informational, neutral

If you see red, prioritise. Amber is not automatically bad — it means "pay attention".
